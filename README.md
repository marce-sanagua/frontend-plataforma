# Frontend · Plataforma de Inscripciones

PP3 — Desarrollo y Aplicación de Sistemas en la Nube · IFTS Nº16

## Descripción

Frontend de la plataforma de inscripciones. Consume el ms-usuarios para autenticación con JWT y gestión de usuarios.

## Funcionalidades

- Login con email y contraseña
- Guarda el token JWT en localStorage
- Lista de usuarios (GET público)
- Crear usuario con token (ruta protegida)
- Muestra mensaje de error si el backend responde 401

## Tecnologías

- HTML, CSS, JavaScript vanilla
- Deploy en Vercel
